# 🎓 Teaching Material

A small, growing collection for teaching non-technical people how apps get built and connected —
using the real **Underlisted** project as the example. (Seed of a possible future teaching project.)

## What's in here

| File | What it is |
|---|---|
| `HOW_IT_CONNECTS.pdf` / `.html` | The plain-English guide: how all the websites (GitHub, Streamlit, Supabase, Netlify, Namecheap, Resend, RentCast) connect, and what API keys are. |
| `TEACHING_SLIDES.pdf` / `.html` | A 9-slide deck of the same lesson — present it, or screen-record it for video. |
| `VIDEO_CLIPS_PLAN.md` | Storyboard + scripts for short teaching video clips (one idea per clip, 30–60 sec). |

## How to use it
1. **Read / print** `HOW_IT_CONNECTS.pdf` to learn the theory.
2. **Present** `TEACHING_SLIDES.pdf` (or open the `.html` full-screen).
3. **Record clips** later using `VIDEO_CLIPS_PLAN.md` as the script.

## To edit & re-make a PDF
The `.html` files are the editable sources. After changing one, regenerate its PDF
(Claude can do this in seconds with the browser-to-PDF step used to create them).
