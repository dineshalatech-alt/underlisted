Data Source Integration Plan

How we connect each source to Underlisted · what I need from you · build order · 2026-06-13

How a source connects, in plain words: every source becomes a small "helper file" that follows the same recipe as our existing ones — fetch the data → map it to our home record → save it in the shared cache. The app only ever reads the cache, so customers never wait and we never pay per page view. Two shapes: (A) API + key (a licensed feed we unlock with a key you grab) and (B) free file/CSV (a public download we refresh on a schedule, no key).
